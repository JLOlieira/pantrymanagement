# Pantry Management App

Bem-vindo ao Pantry Management App! Este projeto é um sistema completo Fullstack para gerenciamento de despensa, compras e organização de produtos alimentícios.

## Visão Geral

O Pantry Management App permite que usuários:

- Cadastrem e gerenciem itens da despensa
- Adicionem produtos ao carrinho de compras
- Realizem login e cadastro de usuários
- Personalizem configurações

## Principais Funcionalidades

- **Autenticação de Usuário:** Login, registro e rotas privadas
- **Gerenciamento de Produtos:** Adição, edição e remoção de itens
- **Carrinho de Compras:** Organização de compras e visualização de itens
- **Interface Responsiva:** UI moderna e adaptável a dispositivos móveis
- **API RESTful:** Backend robusto para manipulação de dados

## Tecnologias Utilizadas

### Frontend

- **React.js** (Vite)
- **JavaScript (ES6+)**
- **CSS Modules**
- **Context API** para gerenciamento de autenticação
- **Vercel** para deploy

### Backend

- **Node.js**
- **Express.js**
- **MongoDB** (banco de dados não relacional)

## Diferenciais

- Sistema Fullstack completo (frontend + backend)
- Código limpo e modularizado
- Componentização reutilizável (botões, inputs, modais, etc.)
- Separação clara entre lógica, apresentação e dados
- Pronto para deploy em ambientes modernos (Vercel, Render, etc.)

## Como Executar Localmente

### Frontend

1. Acesse a pasta do frontend
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
4. Acesse `http://localhost:5173` no navegador

### Backend

1. Acesse a pasta do backend
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Configure as variáveis de ambiente (exemplo: `.env`)
4. Inicie o servidor:
   ```bash
   npm start
   ```
